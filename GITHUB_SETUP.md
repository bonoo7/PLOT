# GitHub

هذا المشروع هو لعبة "الحبكة" - لعبة حفلة تفاعلية لـ 3-10 لاعبين.

تم نشره بنجاح على GitHub

## الفرع الرئيسي: main
- الإصدار المستقر والآمن

## الفروع الإضافية:
- `develop`: الميزات قيد التطوير
- `feature/*`: ميزات جديدة
- `bugfix/*`: إصلاحات الأخطاء

## الملفات المهمة:
- README.md - دليل البدء السريع
- GDD.md - تصميم اللعبة
- TECHNICAL_IMPLEMENTATION.md - التفاصيل التقنية
- ROADMAP.md - خطة العمل المستقبلية
- CONTRIBUTING.md - دليل المساهمة
