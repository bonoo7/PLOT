# تعليمات إنشاء GitHub Personal Access Token

## الخطوة 1: الذهاب إلى إعدادات GitHub
اذهب إلى: https://github.com/settings/tokens

## الخطوة 2: اضغط "Generate new token"
اختر "Generate new token (classic)"

## الخطوة 3: ملء البيانات
- **Note**: plot-game-upload
- **Expiration**: 90 days (أو أكثر)
- **Select scopes**: اختر "repo" (الخيار الأول)

## الخطوة 4: انسخ التوكن
انسخ التوكن الطويل الذي سيظهر (لن يظهر مرة أخرى!)

## الخطوة 5: استخدمه في PowerShell
عند طلب كلمة المرور، الصق التوكن
